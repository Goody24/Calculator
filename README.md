
# Calculator

A simple calculator built with pure HTML, CSS, and JavaScript.
Now supports **Dark Mode toggle**  

## 🌟 Features

- Basic arithmetic operations
- Percentage and sign toggle
- Dark Mode / Light Mode (with saved preference)
- Responsive design
